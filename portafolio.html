Curriculum Vitae

Introducción al desarrollo web: Primer tema del módulo, en el que se practicaron los conceptos básicos de HTML mediante la creación de un currículum vitae.

Tienda de Mangas

Introducción a CSS: Segundo tema del módulo, donde se practicaron las diferentes formas de aplicar CSS a una página HTML. Estas prácticas se implementaron en un proyecto basado en el diseño de una tienda web según nuestra preferencia.

Iguana Pages

Alineando contenido con Flexbox: Tercer tema del módulo, donde se aprendió a utilizar Flexbox para facilitar la alineación de contenido en CSS. Las prácticas incluyeron un desafío en el que se debía realizar una réplica exacta de una página web llamada Iguana Pages.

Tienda Mangas Shop Chile

Framework Bootstrap: Cuarto tema del módulo, donde se aprendió el uso del framework Bootstrap. La práctica se aplicó en el desafío llamado Cuppon, en el cual se podía crear una réplica de una página o diseñar una versión personalizada, siguiendo la estructura indicada.
